Android platformu için geliştirilmiş, kullanıcıların yemek tariflerini kategorilere ayırarak kaydedebileceği ve yönetebileceği bir mobil uygulamadır.

Proje Özeti

BuseProje, Android Studio ve SQLite kullanılarak geliştirilmiş bir tarif defteri uygulamasıdır. Kullanıcılar kendi tariflerini ekleyebilir, kategorilere ayırabilir, tarifleri listeleyebilir, düzenleyebilir ve silebilir. Uygulama, internet bağlantısına ihtiyaç duymadan çalışır ve tüm verileri telefon üzerinde güvenli bir şekilde saklar.

Özellikler

Tarif ekleme
Tarif düzenleme ve silme
Kategori bazlı organizasyon (Kahvaltı, Ana Yemek, Tatlı, İçecek vb.)
Kategoriye göre tarif filtreleme
Tarif detaylarını ayrı ekranda görüntüleme
SQLite ile yerel veri saklama




Kullanılan Teknolojiler

Dil: Java

Platform: Android (Min API 26)

Veritabanı: SQLite

IDE: Android Studio

Arayüz: XML

Kullanım

Tarif Ekle: Tarif adı → Kategori seç → Açıklama yaz → Kaydet

Tarif Görüntüle: Listeden bir tarife tıklayarak detaylarını görüntüleyebilirsiniz

Tarif Güncelle: Tarife tıkla → Düzenle → Kaydet

Tarif Sil: Tarife tıkla → Sil

Kategoriye Göre Filtreleme: Alt menüden bir kategori seçerek o kategoriye ait tarifleri listeleyebilirsiniz
